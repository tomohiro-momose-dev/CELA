# 公式リファレンス・キャッシュ（`docs/refs/`）

ニッチ言語・ベンダー DSL・公式 API の抜粋をここに保存し、エージェントが再検索なしで参照できるようにする。

## 運用

1. 実装前に公式マニュアルを取得する  
2. 構文・例・版注記を `docs/refs/<tool>/` に保存  
3. 各ファイルに **出典 URL** と **取得日** を書く  
4. 以降はローカルを優先。マニュアル更新が疑われるときだけ再取得  

## 例

```text
docs/refs/
  teraterm/
    README.md
    ttl-logging.md
  some_vendor_api/
    README.md
```

## テンプレート（各ツール README）

```markdown
# <tool> 公式リファレンス（ローカル）

- 出典: https://...
- 取得日: YYYY-MM-DD
- 対象バージョン: ...

## 索引

| ファイル | 内容 |
|----------|------|
| ... | ... |
```
